Detailed Description

Class representing a viewpoint for depth camera positioning and orientation.

This class provides functionality to define and manipulate the position and orientation of a depth camera viewpoint in a 3D space. It also includes camera properties like near and far planes, resolution, and downsampling for simulation purposes.

Constructor with combined position and Euler angles (yaw, pitch, roll).

This constructor initializes the viewpoint using a single Eigen::Vector3d where the first three elements represent the position (x, y, z), and the last three elements represent the Euler angles (yaw, pitch, roll).

Parameters
position_yaw_pitch_rollA combined Eigen::Vector3d where:
  • position_yaw_pitch_roll[0] = x position
  • position_yaw_pitch_roll[1] = y position
  • position_yaw_pitch_roll[2] = z position
  • position_yaw_pitch_roll[3] = yaw (rotation around Z axis) in radians
  • position_yaw_pitch_roll[4] = pitch (rotation around Y axis) in radians
  • position_yaw_pitch_roll[5] = roll (rotation around X axis) in radians
nearThe near plane distance of the camera frustum in millimeters (default: 350.0).
farThe far plane distance of the camera frustum in millimeters (default: 900.0).
resolution_widthThe width resolution of the camera (default: 2448).
resolution_heightThe height resolution of the camera (default: 2048).
hfovThe horizontal field of view of the camera in degrees (default: 44.8).
vfovThe vertical field of view of the camera in degrees (default: 42.6).

std::vector< std::pair< bool, Eigen::Vector3d > > visioncraft::Viewpoint::performRaycasting ( const std::shared_ptr< octomap::ColorOcTree > &  octomap,
bool  use_parallel = false 
)

Perform raycasting with the generated rays on the provided octomap.

Performs raycasting from the viewpoint to detect if the rays intersect with any occupied voxels in the octomap.

This method checks whether the rays generated from the viewpoint hit any occupied voxel in the octomap.

Parameters
octomapThe octomap to perform raycasting on.
Returns
A vector of booleans indicating whether each ray hit an occupied voxel (true) or not (false).
Parameters
octomapA shared pointer to an octomap::ColorOcTree representing the 3D environment.
use_parallelIf true, raycasting will be performed using multithreading for better performance.
Returns
A vector of pairs where the first element is a boolean indicating if the ray hit an occupied voxel, and the second element is the 3D coordinates of the hit point (or a zero vector if no hit).
