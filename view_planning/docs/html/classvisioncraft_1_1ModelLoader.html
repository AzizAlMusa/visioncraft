This class provides functionality to load 3D models in various formats using VTK and process them for various applications.

Calculate the average spacing of the point cloud points and set the private variable pointCloudSpacing_.

This function calculates the average spacing of the points in the point cloud by finding the nearest neighbor for each point and averaging the distances. The calculated average spacing is then set to the private variable pointCloudSpacing_.
